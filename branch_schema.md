# Branchschema
## Namensregeln
- Hauptbranch: main
- Feature Branch: Nachname_Feature
- Erkennbarkeit der Arbeit: Immer nur eine Person Pro Branch

## Arbeitsregeln
- Direkt auf main Arbeiten: Man kann auf main Arbeiten wenn es nicht zu komplex ist
    - Durch die Komplexität kommt es leichter zu Fehlern.
- Erstellung eines neuen Branches: Komplexere Arbeiten
    - Durch die Komplexität kommt es leichter zu Fehlern, mit dem Branch wird geregelt, dass der main Branch nicht zerstört wird.
- Branch Löschen: Machen wir nicht
    - Wäre schlecht wenn man etwas aus dem Branch wieder braucht.
